# Example Outputs

This folder contains repository-safe public examples that show the visual direction of the project.

## Included Examples

- `linux-kernel-system-1.jpg` - detailed English architecture figure example
- `rag-workflow-en.svg` - English AI workflow diagram example
- `industrial-monitoring-zh.svg` - Chinese engineering workflow example with preserved English technical terms
- `benchmark-ablation-panel.svg` - locally rendered quantitative publication figure

## Source Files

- `linux-kernel-system-diagram-prompt.txt` - original prompt for the Linux kernel example
- `rag-workflow-en-prompt.txt` - source prompt for the English workflow diagram
- `industrial-monitoring-zh-prompt.txt` - source prompt for the Chinese engineering workflow diagram
- `benchmark-ablation-request.json` - concise plotting request
- `benchmark-ablation-spec.json` - generated full plotting spec

These examples are safe for a public repository and do not rely on private paper figures.
